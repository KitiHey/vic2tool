# An all in Victoria 2 toolbox built in Qt for modding
Features:
    - Pop distribution
*Is planned to add more features*

# Run
Make sure you have [uv](https://docs.astral.sh/uv/)

## Linux
```bash
uv run setup.py linux run
```

## Windows
> [!WARNING]  
> You can't run on windows this code directly from git, you would need to get the pre-packaged code.
```bash
uv run setup.py windows run
```

## Mac
Not supported
